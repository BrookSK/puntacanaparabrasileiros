# Arquivos de Referência do WordPress

Cole aqui os HTMLs salvos do site WordPress (Ctrl+S no navegador).

## Instruções:
1. Abra cada página no navegador
2. Ctrl+S → Salvar como "Página web, completa"
3. Coloque o .html aqui e a pasta de assets (_files) também

## Páginas para salvar:
- home.html (homepage)
- sobre-nos.html
- contato.html
- experiencias.html (listagem de passeios)
- passeio-detalhe.html (qualquer passeio individual)
- transfer.html
- blog.html
- blog-post.html (qualquer post)
- minha-conta.html
- carrinho.html
- checkout.html
